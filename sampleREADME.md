# My Audio Playlist

> A CRUD and SPA application that allow to add create ,  remove , update , delete , and of course  listen  to your favorite songs.

> Jquery , PHP , Materialize


**Technologies**

- HTML5
- HTML Audio Api
- CSS 
- Media queries
- Materialize
- Fully Responsive
- Javascript
- jQuery
- Ajax
- PHP
- MySql  
- Module Patterns  
- ES6 Features
