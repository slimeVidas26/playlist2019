My Audio Playlist

A CRUD and SPA application that allow to add create , remove , update , delete , and of course listen to your favorite songs.

Technologies used on this project.

  • HTML5
  • CSS and CSS3
  • media Queries
  • Materialize
  • JavaScript
  • Module Patterns
  • jQuery
  • PHP
  • MySQL
  • HTML Audio API
  • ES6 Features
  • Ajax
